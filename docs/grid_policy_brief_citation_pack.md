The first block is the **core pack**; the second block is **optional** if you want to document specific extensions. ## Claims that should stay uncited unless you add a new source

These are good arguments, but I would **not** present them as sourced factual claims unless you add dedicated evidence:

- that triage could redirect **“several billions”** of public support;
- that specific borders are already clearly merchant candidates, unless you name and source them;
- that ACER statistics show exactly **65% fewer CBCA decisions**, unless you insert the primary ACER/Commission source for that exact number;
- that producer-versus-consumer-surplus conflicts are driving a given country’s position in a specific corridor, unless you have a direct source.
